## Instrucciones
### Parte 1: Representación de los problemas con Desmos
Deberás ingresar las siguientes restricciones y función objetivo en Desmos para cada uno de los siguientes problemas. Para cada caso:

* Escribe las restricciones como desigualdades.
* Incluye la recta de isoutilidad función objetivo escrita como igualdad, por ejemplo $2x+4y=Z$ y ajusta Z para observar el comportamiento.
* Analiza visualmente la región factible, si existe.
* Clasifica el tipo de solución del problema.

#### Problema 1
* Maximizar: $$20 X + 14 Y$$

Sujeto a:
* \\(2 X + 3 Y ≤ 35\\)
* \\(1.5 X + 1.9 Y ≤ 18\\)
* \\(2 X + 7 Y ≤ 40\\)
* \\(X, Y ≥ 0\\)

#### Problema 2
* Maximizar \\(3 X + 2 Y\\)

Sujeto a
* \\(2 X + Y ≤ 2\\)
* \\(3 X + 4 Y ≥ 12\\)
* \\(X, Y ≥ 0\\)

#### Problema 3
* Maximizar \\( 150 X + 300 Y\\)

Sujeto a:
* \\(8 X + 2 Y ≥ 16\\)
* \\(X + Y ≥ 5\\)
* \\(2 X + 7 Y ≥ 20\\)
* \\(X, Y ≥ 0\\)

#### Problema 4
* Maximizar: \\(2 X + 4 Y\\)

Sujeto a: 
* \\( X + 2 Y ≤ 5 \\)
* \\( X + Y ≤ 4\\)
* \\(X, Y ≥ 0\\)

### Parte 2: Análisis detallado del problema con solución óptima única

A partir de la gráfica obtenida en Desmos, realiza lo siguiente:

1. Identifica visualmente la región factible
2. Determina los puntos extremos de la región factible.
3. Evalúa la función objetivo en cada punto extremo para identificar la solución óptima.
4. Dibuja la recta de isoutilidad (línea de nivel de la función objetivo) que pasa por el punto óptimo.
5. Describe su dirección de optimalidad (por ejemplo: dirección hacia donde se maximiza el valor de Z).
